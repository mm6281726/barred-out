```
* c - increase # of columns
* C - decrease # of columns
* r - increase # of rows
* R - decrease # of rows
* o - increase oscillation speed
* O - descrease oscillation speed
* s - increase microphone gain sensitivity
* S - decrease microphone gain sensitivity
* d - increase sensitivity depth
* D - decrease sensitivity depth
* i x 0 - no image
* i x 1 - static image
* i x 2 - image modulation
* I - invert image shading
* t - increase image threshold
* T - descrease image threshold
* u - increase image modulation speed
* U - decrease image modulation speed
* p - change image
```
